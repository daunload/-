# C# 기본 문법 이해하기

### 이스케이프 시퀀스

- C#은 WriteLine() 메서드에서 사용할 확장 문자를 제공하는데, 이를 이스케이프 시퀀스라고 함
- 역슬래시 기호와 특정 문자를 조합하면 특별한 기능을 제공한다

- \n: 줄 바꿈
- \t: 탭 들여쓰기
- \r: 캐리지 리턴
- \': 작은 따옴표 문자 하나 출력
- \": 큰 따옴표 문자 하나 출력

### 자리 표시자

- {n} 형태로 자리를 만들고, 그 다음에 있는 값을 차례로 넘겨받아 출력

```c#
Console.writeLine("{0}", "Hello, C#");
// Hello, C#


Console.writeLine("{0}, {0}", "Hello, C#");
// Hello, C#, Hello, C#
```
